## 🔍 如何查看字段映射

1. 选用模板后，进入"我的模板"
2. 点击模板查看"详情"
3. 在"详细内容"区域会显示：
   ```
   订单编号：{{character_string1.DATA}}
   当前状态：{{thing2.DATA}}
   ```
4. 记录字段名：
   - `character_string1` = 订单编号
   - `thing2` = 当前状态

## ⚠️ 重要提示

1. **模板ID是28位字符串**，类似：`RWj4PBhIbcTzunNXYo-jJPTw9P_S7HQGCtqmDQTcqWg`
2. **字段名必须准确**，如 `thing1`, `thing2`, `character_string3` 等
3. **每个模板都要截图保存**
4. **只能申请5个模板**，所以要谨慎选择
